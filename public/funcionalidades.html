Funcionalidades

A Orchid é a solução ideal para monitorar a luminosidade das orquídeas, oferecendo um sistema confiável e intuitivo com tecnologia avançada para melhorar a produção.


Monitoramento em tempo real

→ Dados atualizados direto do sensor, sem atrasos.

Histórico de dados

→ Consulte registros diários, semanais e mensais da luminosidade.

Alertas Inteligentes

→ Notificações quando a luz estiver fora do ideal.

Integração de outros sistemas

→ Possibilidade de conectar sensores de temperatura e umidade futuramente.

Relatórios de produtividade

→ Comparação antes e depois do uso do sistema.

Suporte técnico 24/7

→ Assistência sempre disponível para dúvidas ou problemas.

Acesso fácil e seguro

→ Login de qualquer dispositivo com segurança.

Modo multiusuário

→ Permite que diferentes funcionários tenham acesso ao sistema com permissões personalizadas.

Configuração personalizada

→ Ajuste os níveis de luz para cada espécie.

Exportação de relatórios

→ Baixe relatórios em PDF ou Excel para análises detalhadas.
