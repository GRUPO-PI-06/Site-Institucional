Perguntas Frequentes
Dúvidas? Respondemos para você!

Como a Orchid monitora a luz das estufas?
A Orchid usa sensores de luminosidade conectados a um sistema em tempo real que registra e analisa a luz recebida pelas estufas.
O sistema funciona para qualquer tipo de orquídea?
Atualmente, o sistema é compatível com as espécies Phalaenopsis, Vanda e Catleya, que são as mais comuns em cultivo profissional.
Preciso de conhecimentos técnicos para usar o Orchid?
Não. A plataforma foi desenvolvida com uma interface intuitiva e amigável para qualquer usuário.
O Orchid pode ser usado em mais de uma estufa?
Sim. Você pode monitorar múltiplas estufas simultaneamente através de uma única conta.
O sistema envia notificações automáticas?
Sim! Você recebe alertas instantâneos por e-mail caso algo saia dos parâmetros ideais.
